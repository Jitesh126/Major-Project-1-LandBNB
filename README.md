# Major Project 1 - LandBNB
## Link to connect - https://major-project-1-landbnb.onrender.com

Full stack MERN stack project (React not been used)  
Airbnb Clone  
MVC Design Pattern  
Authentication && Authorization implemented using passport-local-strategy

## How to run this project on your local machine
1 -->  (npm install) write this line in terminal  

2 -->  create .env file  

3 -->  create your account on [cloudinary](https://cloudinary.com/)  

4 -->  store cloud_name, api_key, api_secret in .env file Like -->  
CLOUD_NAME=...  
CLOUD_API_KEY=...  
CLOUD_API_SECRET=...  

5 --> create your account on [mapbox](https://www.mapbox.com/)  

6 --> store default_public_token in .env file Like -->  
MAP_TOKEN=...  

7 --> store SECRET=... in .env file (value can be any string)  

8 --> create your account on MongoDB Atlas, create Cluster, copy connection link of Cluster  

9 --> store ATLASDB_URL=paste the connection link here  

10 --> Replace <db_password> with the password for the database user in ATLASDB_URL.
